# evoflare.app

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```

### Run your unit tests
```
npm run test:unit
```

### Run on heroku (set env)
```
heroku config:set VUE_APP_ROOT_API=https://evoflare-api.herokuapp.com/
```
